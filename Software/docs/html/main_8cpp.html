Detailed Description

Main file for the environmental sensor dashboard.

Reads data from BME680, LTR390, and VCNL4040 sensors and displays values on a TFT screen. Handles touch input to switch between main and detail pages. Implements automatic display brightness adjustment based on ambient light and hand proximity.
